# Criar uma api TO-DO List para gerenciamento de tarefas

1. O user pode adicionar uma nova tarefa;
2. Editar o nome e a descricao de uma tarefa;
3. Listar todas tarefas;
4. Listar as terefas nao concluidas.

## Passos para o desenvolvimento:

1. Importar bibliotecas;
2. Criar uma instancia da classe FastAPI para definição de rotas da aplicação, associando funções a diferentes rotas HTTP e facilitando a execução do server.
3. Criar os modelos através do BaseModel do Pydantic, para validação de dados;
4. Conectar à DataBase pelo SQLAlchemy;
5. Criar as rotas e funções associadas.
